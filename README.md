# DIGI_NEWS
This is a multipurpose website template, including pages
- Main Page
- contact us
- about us
- News
- Movies and series
- log in
- Register
- forgot password
- Welcome
- not found (404)
- profile

## Made with
### Bootstrap
### UI kit
### jQuery

## Options
### Full Responsive
### Completely Persian
### Right Align
